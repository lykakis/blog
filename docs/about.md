# About me

I'm Pavlos Lykakis, a software engineer based in Heraklion, Crete.

I work at [Cytech Mobile](https://www.cytechmobile.com/), mainly [mCore](https://www.cytechmobile.com/mobile/mcore-a2p-wholesale-platform/) and [Sendium](https://github.com/cytechmobile/sendium).

[Sendium](https://github.com/cytechmobile/sendium) is an open-source headless SMS gateway built with Java and Quarkus. I help maintain it.

This site is where I write about my thoughts on things i am learning.

The idea came after i saw Steph Ango [blog](https://stephango.com/). I really liked the simplicity on his writing, I would suggest you to check those three:

[Don't delegate understanding](https://stephango.com/understand)
